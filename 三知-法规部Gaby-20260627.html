SFC 主席黄天佑（Kelvin Wong）与 CEO梁凤仪联合发表 2025-26 年报，主题"Diversified Engines Underpin High-Quality Growth"。核心数据：

  • 二级市场：股票市场平均每日成交额（ADT）+54% 至 HK$2,580 亿——历史新高；IPO 后再融资 +18% 至 HK$2,590 亿；

  • 南向 + 互联互通：南向通 ADT +84% 至 HK$1,241 亿，占港市成交历史最高 24%；

对我们的启示：（1）一级 + 二级共振："IPO 募资 +272% × 南向 ADT +84%"是结构性而非脉冲——基石 / 一级团队应将2026 H2 港股新发节奏假设从"周度跟踪"升级为"滚动管线模型"，把WVR + 18A/18C + 极重大收购视同新上市四类纳入同一统计口径；（2）南向 ADT 历史 24% 占比意味着定价权回归南向 + 港币定价的 alpha 衰减，应同步评估港股通人民币柜台（上周陆家嘴论坛主线）对定价与流动性的实际影响；（3）价差收窄 + 成交速度提升对量化 / 算法交易策略构成结构性利好——二级与量化团队应重估市场冲击与滑点假设。
SFC6 月 23 日（26PR97）：港交所衍生品市场投资者识别制度（HKIDR-DM）咨询总结

SFC 完成HKIDR-DM咨询，9 份反馈一致支持，正式确认推进。核心机制：

  • 适用范围：所有港交所期货合约、期权合约、股票期权的交易层申报；

  • 对接：持牌法团 / 注册机构须将客户姓名 + 身份信息上报至 SFC 中央数据库（与现行证券 OTCR 框架并行）。
对我们的启示：（1）合规 / IT 提早 24 个月布局：2028 Q2 看似遥远，但证券 HKIDR 当年系统改造耗时 18–24 个月——建议本季度发起"HKIDR-DM 系统升级前期评估"，重点核对OMS / 风控 / 客户主数据三大系统对接路径；（2）跨产品监控并轨：HKIDR-DM 一旦运行，SFC 将拿到同一客户在股票 + 衍生品两端的完整画像，跨产品的对盘 / 抢盘 / 联合操纵调查能力进入新代际——交易合规 surveillance 政策需相应升级；（3）客户披露：对QFII / 境外机构 / 家族办公室客户的身份信息收集与跨境流转合规（PDPO + PIPL）须提前梳理。

二、中国证监会 / 国务院 / 央行（CSRC / PBOC）

CSRC6 月 26 日："史上最重私募罚单"——玖瀛资产 + 腾创投资合计罚没近人民币 6,000 万元

深圳证监局发布行政处罚决定，对玖瀛资产管理有限公司与深圳前海腾创投资有限公司合计罚没近人民币 6,000 万元，3 名责任人个人罚没逾 100 万元，实际控制人 5 年市场禁入 + 5 年证券交易禁止——刷新中国私募基金行政处罚纪录。

  • 违规事实：利用私募基金产品向玖瀛资产及关联方输送利益；通过不当债券交易取得超 RMB 2,800 万非法所得；向监管虚假报告；
  • 合规背景：正值《国务院办公厅关于加强监管防范风险促进私募投资基金高质量发展的指导意见》三年行动方案制定阶段；《私募投资基金信息披露监督管理办法》（CSRC 令第 233 号）9 月 1 日生效；
  • 历史背景：2023 年至 2026 年 Q1，CSRC 对 1,805 家私募管理人采取行政监管措施，97 家遭行政处罚——本次为该执法周期内的最重单笔。
对我们的启示：（1）境内私募合规自查：若我司在境内有 PFM / WFOE 通道（含债券 / 商品策略），应立即对照玖瀛 / 腾创违规事实做内部"三防"自查——（i）关联方利益输送（投资标的 / 估值 / 退出价格）；（ii）债券交易公允性（询价 / 对盘 / 成交流程）；（iii）信息披露真实性（基金合同 / 季报 / 重大事项）；（2）令第 233 号倒计时：距 9 月 1 日生效仅 2 个月，七大禁止行为 + 季报 / 临时 / 清算三档披露 + 4 大类 24 项基础信息披露必须在 8 月底前完成 IT 与合规对齐；（3）跨境联动信号：境内"史上最重"也对HK 持牌主体跨境提供境内基金通道构成警示——HK SFC 与 CSRC 已建立双边监管合作协议（MoU），跨境信息共享在 2026 年明显加速。

CSRC令第 233 号倒计时：《私募投资基金信息披露监督管理办法》9 月 1 日生效——首部私募信披行政规章

《办法》7 章 44 条，2026 年 9 月 1 日生效。核心制度：

  • 七大禁止行为：含虚假记载、误导陈述、重大遗漏、单方面披露不当业绩、未按规定披露关联交易等；
  • 三类报告：季度报告 + 临时报告 + 清算报告差异化要求；
  • 信息披露事务管理：明确信息披露事务负责人、台账留存（≥ 10 年）、电子化备案；
  • 覆盖规模：对应境内 23 万亿元私募基金存量。
对我们的启示：对我司境内 PFM / QFLP 通道，建议7–8 月完成"令 233 IT 升级"——三件事：（1）季报 / 临时报告模板升级（重大事项触发条件 + 24/48 小时披露窗口）；（2）"披露事务负责人"任命入岗 + KYC 留痕；（3）10 年台账留存对接（含云存档冷备份）。境外母基金对境内私募合作方，则应在 Side Letter 中加入"合作方须确认令 233 合规"声明，避免穿透责任。

Thomson Reuters 6 月 23 日宣布下一代 CoCounsel Legal 早期接入（early access）开放，是其 2023 年并购 Casetext 后的最大产品迭代。三大新意：

  • Agentic 架构："规划 → 选择工具 → 检索权威内容 → 中途自适应"，定位为"资深 Associate"而非"第一年待指示"——基于 Anthropic Claude Agent SDK构建（与我司主用栈一致）；
  • 上线路径：本周起美国全 CoCounsel Legal 客户可在产品内切换新旧体验；2026 年 8 月美国 GA，加拿大 / 英国 / 澳大利亚分阶段推出。
对我们的启示：（1）"Fiduciary-Grade" 成新基准："每一步可追溯引用 + 客户数据不上训练"将成为所有面向法律合规的 AI 产品的默认门槛——本月内对我司现用的所有 LLM / Agentic 工具做"Fiduciary 三项测试"自查（可追溯引用 / 隔离训练 / 推理可视）；（2）"Claude Agent SDK 共栈"信号：TR 选择 Claude Agent SDK 作为下一代核心，与我司既有"Claude Opus 4.x 主栈"形成同栈协同优势——可启动"CoCounsel 早期接入 + Claude Agent SDK 自建 agent"双线评估，本季度形成结论；（3）对外聘律所采购的反向影响：未来若外聘律所大量采用 CoCounsel 新版，我司应在 Engagement Letter 中要求"AI 工具使用披露 + 输出可追溯证据保留"作为账单审核基础，避免支付"由 AI 直接生成 + 律师挂名审阅"的高单价工时。
Legal AIA&O Shearman + Harvey "Agentic Agents" 首批落地——反垄断申报 / 网络安全 / 基金设立 / 贷款审查

A&O Shearman 与 Harvey 联合披露面向复杂法律工作流的 agentic AI agents商用计划：首批将覆盖反垄断申报分析（HSR/EU MC / 中国市场监管）、网络安全合规审计、基金设立（PE/VC fund formation）、贷款审查四类高频高价值任务。Agents 内置 A&O 多年沉淀的法律专业知识，"分解 → 检索 → 多步推理 → 综合"形成完整工作产品，内部使用 + 对外销售（含其他律所）。
